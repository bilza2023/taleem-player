Minimal Example (Gold Deck)

The gold deck is the reference implementation for Taleem.

It demonstrates correct structure, spacing, typography, image usage, and slide flow.

This deck is intentionally simple. It avoids edge cases and advanced features.

Any change to the core system is validated against the gold deck.

If the gold deck renders correctly, the system is considered stable.

This deck should be the first thing new users study.

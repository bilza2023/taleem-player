How to Run the Demo

The demo project is a static site that loads a JSON deck and renders it using taleem-browser.

No build step is required. The demo can be opened directly in a browser or deployed via GitHub Pages.

Multiple demo pages exist to showcase navigation, autoplay, and range-based control.

These demos double as executable documentation.

If a demo works, the system works.

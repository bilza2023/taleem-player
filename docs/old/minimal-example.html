Minimal Example

This page describes the smallest complete example of a Taleem presentation.

The goal is not visual polish. The goal is correctness.

What the Minimal Example Contains

The minimal example uses the Gold Deck.

It includes:

  • a valid Schema v1 deck
  • all supported slide types
  • no custom styling
  • no application logic

Why This Matters

If the minimal example renders correctly:

  • the schema is valid
  • the renderer works
  • the engine behaves correctly

This makes it the fastest way to verify the health of the system.

What This Page Is Not

This is not a tutorial.

It does not explain how to author content or build applications.

It exists to prove that the system runs.

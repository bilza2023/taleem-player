taleem-browser

taleem-browser is the index-based engine in the Taleem system.

It displays slides by position. One slide at a time. No notion of time.

What the Browser Engine Does

taleem-browser:

  • accepts rendered HTML slides
  • tracks a numeric slide index
  • shows exactly one slide at a time

Moving forward or backward changes the index.

No Time Awareness

The browser engine has no concept of:

  • seconds
  • durations
  • timelines
  • playback speed

It does not autoplay. It does not schedule transitions.

Slides Are Opaque

taleem-browser does not inspect slide content.

It does not:

  • parse slide structure
  • read data attributes
  • interpret slide meaning

A slide is treated as an opaque HTML unit.

Typical Uses

The browser engine is suited for:

  • manual presentations
  • document-style navigation
  • slide browsing and review
  • editor previews

What Does Not Belong Here

taleem-browser must not:

  • reason about time
  • trigger animations
  • coordinate with audio
  • modify slide markup

If a feature needs time, it belongs in the player.

The Boundary

taleem-browser answers one question:

“Which slide should be visible right now?”

Nothing more.
